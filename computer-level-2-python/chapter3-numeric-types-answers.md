## 一、选择题

C A A A C A C D A C

### 解析：

1. 对于整数类型，输出格式包括如下6种。
   - b: 输出整数的二进制方式；
   - c: 输出整数对应的 Unicode 字符；
   - d: 输出整数的十进制方式；
   - o: 输出整数的八进制方式；
   - x: 输出整数的小写十六进制方式；
   - X: 输出整数的大写十六进制方式；
